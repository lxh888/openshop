##### 全局赋值函数：
~~~
printexit([mixed $data = NULL][mixed $data1 = NULL][...]);
~~~

##### 继承框架类：
~~~
framework\cao::printexit([mixed $data = NULL][mixed $data1 = NULL][...]);
~~~

| 参数类型  |  参数名称  |  参数备注  |   是否必须   |  传参顺序   |
| --- | --- | --- | --- | --- |
|  mixed  |  $data  |  要打印的数据  |  否  |  mixed[0]  |
|  mixed  |  ......  |  ......  |  ......  |  mixed[n]  |


- 该方法可以打印多个数据，并且程序会被终止。



```
$arr = array('人生至古谁无死', '留取丹心照汗青');
$str = '往事随风难追忆~';
printexit($arr, $str);
```
